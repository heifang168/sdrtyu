# ALEO POWER Static Website

This repository contains the ALEO POWER B2B independent website and landing pages.

## Main Entry Points

- `index.html` - homepage
- `contact/index.html` - inquiry form page
- `lp/google-ads-b2b-minimal/index.html` - minimal Google Ads B2B landing page
- `thank-you/index.html` - form submission thank-you page

## Inquiry Email

Website forms submit through FormSubmit to:

```text
ge50613386@gmail.com
```

The mailbox must stay activated in FormSubmit for delivery to work.

## Deployment

This is a static HTML/CSS/JS site. Vercel can deploy it directly from the repository root.

Generated reports and local deployment state are intentionally ignored:

- `.vercel/`
- `outputs/`
- `node_modules/`
